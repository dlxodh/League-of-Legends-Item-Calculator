Calculates an item's gold efficiency 아이템 가성비 계산기

Supported Language: Korean and English

주의사항

현재, 빛나는 티끌 아이템이 그냥 구려서 가속이 붙은 아이템들 스탯가가 전체적으로 높게 보임 이는 티끌을 사용한 다이소 템트리의 등장을 막기 위해 의도된 사항으로 보이나, 그로인해 가속이 들어간 아이템의 해석에 주의가 요구됨 상위 아이템 계산에 모두 영향을 끼침

장점

간편하게 가성비 계산 가능 단점
기준 아이템의 가성비에 따라 계산이 달라짐 ex)롱소드가 너프 받을 경우
챔프별 스탯 효율이 다름
가성비가 좋다 = 아이템이 좋다로 연결되지는 않음(옵션의 가치가 다르기 때문)
퍼센트 이속, 생명력 흡수 등의 일부 능력치는 기준 선정의 어려움으로 스탯 가격에 포함 x
